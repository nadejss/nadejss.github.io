Shiru Terms of Service

Last Updated: 4/13/23

Proper Usage

Shiru is a multipurpose and multilanguage bot. You may use these commands as you would like within your server as long as what you are doing does not break discord's TOS.

Shiru stores channelIDs, guildIDs and userIDs. We will ONLY store said IDs if the bot requires it to complete actions within your server. If you would like to delete your data, disable the systems that store it. You can also run commands to or join our support server to request personal data deletion.

READ THIS IF YOU HAVE THE PHONE SYSTEM

Shiru collects messages so they can be sent to the other servers. If you want Shiru to stop collecting messages, disable the system with /set-phone disable

Consider checking this page frequently, as our TOS may change.
